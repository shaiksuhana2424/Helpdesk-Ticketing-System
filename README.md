# Helpdesk Ticketing System

## Overview
The Helpdesk Ticketing System is a Java-based application designed to manage and track support tickets efficiently. The system enables users to create, monitor, and manage support requests while helping organizations streamline incident handling and technical support operations.

## Features
- Create and manage support tickets
- Track ticket status and progress
- Organize support requests efficiently
- Maintain structured ticket information
- Improve issue resolution workflow
- Support database-driven ticket management

## Technologies Used
- Java
- SQL
- MySQL
- HTML5

## Project Structure
- HelpdeskApplication.java – Main application entry point
- Ticket.java – Ticket model/entity
- TicketController.java – Handles ticket-related operations
- TicketRepository.java – Manages database interactions

## Key Highlights
- Designed a unified incident handling interface to prioritize incoming support tickets.
- Structured support workflows to manage issue tracking and resolution processes.
- Implemented database-driven ticket management using SQL and MySQL.
- Organized ticket data to improve monitoring and support operations.
